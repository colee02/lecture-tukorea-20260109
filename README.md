# lecture-tukorea-20260109
한국공학대학교 특강 실습
### 실습 프로젝트 다운로드
1. 오른쪽 상단 *Code* 버튼 클릭 
2. 프로젝트 다운로드
    - *Clone* 주소를 복사해 git으로 내려받거나
    - 또는 *Download ZIP* 메뉴를 클릭

# 실습 #1: 기계학습
### 실습 폴더로 이동
```bash
cd [실습 프로젝트 경로]/tutorial01_ML 
```
### 윈도 PC에 실습 환경 준비
```bash 
python -m venv venv_tutorial01
venv_tutorial01\Scripts\activate 
pip install -r requirements.txt
```

### 실습 코드 실행
```bash
python tutorial_code.py
```

### 실습 종료후 환경을 삭제하려면
1. `deactivate` 명령을 실행해 환경 종료
2. venv_tutorial01 폴더 삭제
